# Vinux-AI
Vinux AI — это независимый локальный гибридный ИИ-чат-бот, построенный по модульному принципу. Проект сочетает в себе быстрое ядро классификации намерений (интентов) и собственную генеративную нейросеть-Трансформер (в разработке), работающую полностью автономно без использования сторонних облачных API.
